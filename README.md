# TOP Investment

TOP Investment is a modern investment web application built with HTML, CSS, JavaScript, and Firebase.

## Features

- User Registration
- User Login
- Secure Authentication
- Dashboard
- Wallet Balance
- Deposit Requests
- Withdrawal Requests
- Transaction History
- User Profile
- Admin Panel
- Firebase Firestore Database

## Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Firebase Authentication
- Cloud Firestore
- GitHub Pages
